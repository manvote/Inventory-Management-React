# Inventory Management System

A simple, vibrant Inventory Management System built using **React**, **HTML**, **CSS**, and **LocalStorage**. This project allows you to manage products, track stock, and monitor low-stock items without any backend or API.

---

## Features

* Add new product details (name, price, quantity)
* Edit existing product details
* Delete products from the inventory
* Stock in / Stock out functionality
* Instant UI updates without page reload
* Highlight low-stock products
* Search products by name
* Dashboard showing **Total Products** and **Low Stock** count
* Separate page for viewing only low-stock products
* Persistent storage using **LocalStorage**
* Dark violet, modern, and user-friendly interface

## Installation

1. Clone the repository:

```bash
git clone <your-repo-url><img width="1919" height="1079" alt="Screenshot 2025-12-17 220605" src="https://github.com/user-attachments/assets/bf39f06c-1a64-4f87-b179-61c6215bdd43" />

```

2. Navigate to the project folder:

```bash
cd inventory-app
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm start
```

The app should now be running at `http://localhost:3000`.

---

## Usage

* Click **Total Products** or **Low Stock** cards to navigate between inventory and low-stock views.
* Use the form to add or update products.
* Search products using the search input.
* Click `+` or `-` buttons to update stock quantity.
